 Calculadora (POO - Java)

Este projeto foi desenvolvido como exercício prático da disciplina de Programação Orientada a Objetos, com o objetivo de aplicar conceitos fundamentais da linguagem Java.

 Conceitos utilizados
Programação Orientada a Objetos (POO)
Criação e uso de classes e métodos
Encapsulamento
Tratamento de exceções (try/catch e throw)
Uso de tipos numéricos (Double, BigDecimal)
Estruturas de decisão (switch-case)
Interação com o usuário via terminal

 Funcionalidades
Operações matemáticas básicas
Validação de entradas (ex: divisão por zero)
Execução via menu interativo no terminal
