# Contribuindo

Obrigado por considerar contribuir.

## Como contribuir

1. Abra uma issue descrevendo a melhoria ou correção.
2. Crie um fork e uma branch para sua alteração.
3. Envie um pull request descrevendo o que mudou.

## Padrões

- Mantenha a organização em `src/main/java/poo/exercicios`.
- Prefira nomes de classe claros e objetivos.
- Inclua exemplos de execução quando adicionar novos exercícios.
